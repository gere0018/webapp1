The Application

This text is a made up description for my fictional application:
Rescue will send a message to your emergency contact when you happen to be in danger. Therfore the app requires access to your location. You will also need to fix an emergency contact that will be accessed when needed.


This App is a web application designed from scratch using only html5 css3 and javaScript without the help of any framework like bootstrap or libraries like JQuery or JQuery mobile. It's an opportunity to review all the concepts studied so far and to practice single page applications in html. Most importantly the Cordova phone gap is used again to allow us to make the app fucntional on Android.
